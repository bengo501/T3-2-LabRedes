Laboratório de Redes de Computadores - Aula 12
Atividade: Roteamento Estático

Descrição:
- Topologia com 5 roteadores, 5 switches e 10 hosts (VPCS).
- Cada roteador conectado a um switch e dois hosts.
- Roteamento estático configurado para garantir conectividade total entre todas as subredes.

Arquivos de configuração:
- R1_config.txt ... R5_config.txt: comandos de configuração dos roteadores.
- VPCS_config.txt: comandos de configuração dos hosts.
- tabela_enderecamento.txt: tabela de endereçamento IP.

Testes realizados:
- Ping entre hosts de diferentes subredes.
- Traceroute para verificar o caminho dos pacotes.

Resultados:
- Todos os hosts conseguem se comunicar entre si.
- Roteamento estático funcionando conforme esperado.
